# Automation Empire 


## 📊 Automated Daily Data Tracker
This is an automated multi-purpose bot that tracks weather, daily jokes, and inspirational quotes using GitHub Actions.

### ✨ Features:
- ⛅ **Weather Sync:** Tracks Dhaka & Cumilla temperature.
- 🎭 **Daily Jokes:** Fetches random jokes to the log.
- 💡 **Daily Quotes:** Collects inspirational thoughts.
- 🕒 **Automated Logging:** Syncs data 16 times a day.
- ☁️ **Cloud Native:** Runs entirely on GitHub Actions.
